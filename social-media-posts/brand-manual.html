03

Logo Usage

Primary Logo

The white logo version is used exclusively on dark backgrounds. Always maintain proper spacing and never distort the proportions.

Logo Sizing

Context Size Position
Hero/Feature Post 280px - 380px Center, prominent
Standard Feed Post 130px - 150px Top right corner
Story Post 380px Center, with drop shadow
Campaign Badge Post 130px Top right corner

Clear Space

Maintain a minimum clear space around the logo equal to the height of the "K" in "Karavane". No other elements should intrude into this space.

Important: Never place the logo on light backgrounds without using a dark container or the colored logo variant.

Logo File Reference

/* Logo file for social media posts */
src: "logo-white.png";

/* Recommended CSS for logo */
.logo {
  width: 150px;
  height: 150px;
  object-fit: contain;
}
04

Post Formats

Feed Posts (Square)

1080 x 1080
pixels

Instagram & Facebook Feed

  • Aspect ratio: 1:1 (square)
  • Standard canvas for all service posts, promotions, and informational content
  • 60px padding on left/right edges
  • 50px margin from bottom for contact section
  • Top accent bar: 12px height
  • Bottom accent bar: 8px height

Story Posts (Vertical)

1080 x 1920
pixels

Instagram & Facebook Stories

  • Aspect ratio: 9:16 (vertical)
  • Used for brand awareness and adventure-themed content
  • More creative freedom with sunset gradients and decorative elements
  • Larger logo (380px) positioned centrally
  • Contact section at bottom with 80px margin

Layout Grid

All feed posts follow a consistent vertical structure:

Section Position (from top) Height/Content
Top Accent Bar 0px 12px
Logo/Badge Area 35px - 60px Variable
Main Content 140px - 280px Variable
Contact Section Bottom - 50px ~100px
Bottom Accent Bar Bottom 8px
Tip: Use absolute positioning for all major sections to ensure pixel-perfect placement across all posts.

Background Patterns

Subtle background patterns add texture without overwhelming content. Always use low opacity (3-8%).

Pattern Opacity Usage
Diagonal Stripes 3% Campaign/pricing posts
Topographic Circles 8% Service/repair posts
Road Lines 5% Checklist/travel posts

Glassmorphism Effect

Used for contact bars and floating cards to create depth and modern aesthetic.

.contact-bar {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  border-radius: 16px;
  border: 1px solid rgba(255, 255, 255, 0.1);
}

Headlines follow a consistent structure with the key service/feature highlighted in orange.

[Adjective] [Noun] Single Braked Axle Service

Export Settings

Posts are exported to PNG using Puppeteer with these settings:

// Puppeteer export configuration
deviceScaleFactor: 1,
type: 'png',
clip: {
  x: 0,
  y: 0,
  width: 1080, // or 1920 for stories
  height: 1080 // or 1920 for stories
}
File Naming: Use the pattern ##-descriptive-name.html for HTML source files and matching .png for exports. Examples: 01-caravan-service-repairs.html, 15-service-packages-promo.html
